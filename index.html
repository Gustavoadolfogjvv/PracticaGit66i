git add . Prepara los archivos (etapa Staged) para ser comitiados (confirmar los cambios)
git commit -m "mensaje de Commit" confirma los cambios que estaban en preparacion (en staget)
git commit -am "mensaje de Commit" Agrega y confirma los cambios que estaban en preparacion (en staget) en un solo comando
git reset --soft HEAD~1 Deshace el ultimo commit de mi repo local (no Publicado) manteniendo los cambios
git commit --amend -m "Este es el mensaje correcto" Arregla el mensaje que has usado para el ultimo commit
git log Listar los commits que tengo en mi historial
git branch nombreRama te crea una nueva rama con el nombre especifico
git branch lista las ramas que tengo en mi repo local
git checkout NombreRama Saltar a la rama especifica
